# Starter Kits

Codebase di repository ini sudah disetup dengan laravel 12 + vue menggunakan laravel starter kits.

## Persiapan
Sudah terinstall:
1. composer
2. php min. version 8.3
3. nvm terbaru

## Local Development
1. Clone repository ini
2. Jalankan
```
composer install
npm install && npm run build
composer run dev
```
